0. Stažení pythonu

Pokud nemáš stažený python, celý návod najdeš tady:

www.python.org/...

Kdykoliv si nebudeš vědět rady, neboj se googlit.

To nejdůležitější, co bys se měl/a během tohoto kalendáře naučit, je řešit problémy, zkoušet, co tě napadne, přicházet na to, proč něco nefunguje, a vyhledávat informace.

S těmito skilly se poté můžeš v programování zlepšovat sám/sama bez průvodce.

1. Jak to zapnout?

Otevři příkazovou řádku (vyhledat ji můžeš jako cmd).
Napiš python (případně python3) do příkazové řádky a zmáčkni enter.
Pokud se ti zobrazí uvítání do Pythonu a nový řádek se třemi šipkami:

>>>

tak jsi právě spustil/a Python.

A můžeš spustit svůj první program.
do příkazové řádky napiš:

>>> print("Hello, world!")

a poté zmáčkni enter (tím začne python kód zpracovávat).

Pokud ti vyskočil nějaký error, zkontroluj, jestli si vše opsal/a zprávně.
Tři šipky a první mezeru neopisuj. Pokud ti program odpoví, tak máš vyhráno.

Pro odchod z pythonu napiš:

>>> quit()

2. Proměnné

Proměnná je jako krabička, ve které se může něco skrývat.
Začneme tím, že do krabiček budeme házet čísla.

Nadefinuj si nějaké proměnné:

nazev_promenne = celé číslo (datový typ "integer")
nazev_promenne2 = desetinné číslo (datový typ "float")
# desetinná čísla píšeme s tečkou: 12.024 nebo 0.021

např.

x = 5

Když napíšeš název proměnné do příkazové řádky, vrátí ti její hodnotu:

>>> x
5        # výstup (output), který ti vrátí python

3. Matika

Zkus experimentovat se znaky:

+, -, *, **, /, //, %

a to tím způsobem, že napíšeš:

číslo nebo název proměnné znak číslo/název proměnné
a stiskneš Enter.

např.:

>>> 21 + x

Dál je můžeš kombinovat a přidat závorky, např.:

>>> (5 + x * 2) % 2

4. Pravdivost

Dnes zkus experimentovat s těmito znaky:

==, !=, <, >, <=, >=

a to způsobem, že napíšeš:

číslo znak číslo
a stiskneš Enter
Místo čísel můžeš používat názvy proměnných.

např.

>>> 7 != 9

# mezery psát nemusíš, ale doporučujeme je psát pro přehlednost.
Tyhle znaky vrací pravdivost výroku, True/False (datový typ "boolean").

Dál je můžeš kombinovat s tím, co už znáš:

>>> x % 2 == 0

nebo

>>> 2 > (5 + (3 + y / 3))

5. Text

Do proměnné můžeme vložit i text ("string").
String je datový typ, který se píše do uvozovek: "nějaký text" nebo 'jiný text'.

Proměnnou můžeme definovat takhle:

nazev_promenne = string

např.

>>> bertik = 'pes'
>>> micka = "domácí mazlíček"

Vypsat obsah proměnné můžeš dvěma způsoby.
Buď napíšeš do terminálu její jméno:

>>> bertik

nebo použiješ funkci print():

>>> print(bertik)

6. Funkce print()

Funkci print() můžeš použít pro vypsání více proměnných v jednom řádku:

>>> print(bertik, "je mé oblíbené zvíře, máme jich doma", x)
# proměnné bertik a x musíš mít předem definované.

>>> print("Doma na mě čeká " + bertik + "a můj" + micka + "Micka")

Pokud chceš vypsat kombinaci textu a proměnných, je dobré použít tzv. f-string:

>>> print(f"Můj nejoblíbenější {micka} je {bertik}, mám jich doma {x - 1}.")

7. Funkce input()

Vstup ("input") jsou data, která počítač dostává od uživatele.
Je to stejné, jako když se zeptáš kamaráda na otázku a on ti odpoví; jeho odpověď je pro tebe vstup.

Vyzkoušet si to můžeš takhle:

>>> answer = input("Jak se máš? ")
>>> print('Máš se: ' + answer)

>>> age = input('Kolik ti je? ')
>>> print(f"Narodil/a si se v roce {2023 - int(age)} nebo o rok dříve.")

# funkce int() mění string obcahující pouze číslice v číslo.

8. Podmínky - if

If vezme podmínku a ověří, jestli je pravdivá. Pokud pravdivá je, nechá provést část programu.
Např. Jestli prší, vezmu si deštník.

if prší:
    vezmu si deštník

Na čem si to vyzkoušet?

>>> age = int(input("Kolik ti je? "))
>>> if age < 17:
...     print("Mazej domů, vždyť už máš dávno po večerce!")
... elif age == 17:
...     print('No dobře, ale ať tě nikdo nevidí.')
... elif age == 18:
...     print('Běž, ale žádný blbosti!')
... else age >= 18:
...     print("Běž se bavit.")

Jestli si pamatuješ, jak si 4. den zkoušel/a znaky <, ==, >= atd., tak ty ti vracely hodnotu True nebo False.
If tuhle informaci (True, False) vezme, a pokud je pravdivá, provede co má zadáno. Pokud není pravdivá, posune se dál na další elif nebo else.

Akce, které se mají provést, když je podmínka splněna, odsazujeme tabulátorem.

9. Funkce help()

Pokud máš spuštěný python, můžeš použít funkci help() k vyhledávání informací.

>>> help()

Otevře se ti uvítání a návod na prozkoumávání python dokumentací.
Teď si můžeš vyhledat nějaký výraz, který už znáš:

>help if

a otevře se ti dokumentace ifu.

Pokud python žádnou dokumentaci nenajde, můžeš se podívat na jeden z výrazů, které ti nabízí, např.:

>help keywords

Zobrazí se ti jeden ze seznamů výrazů, které mají svoji dokumentaci.
Z těch si můžeš nějaké známé vybrat a vyhledat:

>help and

Rozhodně ale nedoporučujeme číst vše a celé :)

10. soubor.py

13. Random

Python má spoustu rozšíření ve formě knihoven.
O různých z nich si můžeš přečíst na internetu.
My si ukážeme knihovnu random a z ní jednu konkrétní funkci randint().
Na začátek kódu napiš:

from random import randint

To ti umožní používat funkci randint z knihovny random, která po zavolání vrací náhodné číslo v rozpětí, které jí zadáš.
Dále v kódu ji pak zavoláš takto:

randint(0, 10)
#vrátí náhodné číslo od 0 do 10

Celý tvůj program pak může vypadat třeba takhle:

from random import randint

if input("hádej číslo od 1 do 3: ") == randint(1, 5):
#pokud si hrajeme na sazku, tak se tu nic podezřelého neděje :D
    print("vyhrál si")
else:
    print("smůla, naval prachy")

14. While cyklus

Dostáváme se k cyklům.
Ty se hodí, když chceš něco dělat opakovaně a nechceš program spouštět znovu a znovu.
Cyklus while funguje velmi podobně jako if, s tím rozdílem, že se opakuje stále dokola, dokud podmínka platí. Pokud do podmínky napíšeme např. True nebo 1, tak cyklus poběží do nekonečna.
Můžeme si to vyzkoušet na tomto programu:

while input("Co mi povíš? "):
#tohle je tvůj nejlepší kamarád, bude se smát jakémukoliv tvému vtipu, dokud se s ním budeš chtít bavit
    print("XD\n")
else:    #i tady můžeš použít else, ale nemusíš
    print(":(")

15. Příkazy break, continue

Příkaz break ukončí cyklus a continue nás vrátí na jeho začátek. Ukázat si to můžeme na včerejším programu a na našem nejlepším kamarádovi:

counter = 30
joke = ""

while joke != "nenávidím tě":
    if not joke:
        print(":|")
        continue
    print("XD\n")
    if counter < 0:
        print("Už smíchy nemůžu dýchat... x_x")
        break
    counter -= 1
    # je to stejné jako by si napsal/a counter = counter - 1
else:
    print(":(")

16. Seznamy

Z datových typů už známe integer, float, string a boolean (True/False).
Další užitečný je seznam ("list").

Vytvoříš ho takhle:

nazev_seznamu = []    # tohle je prázdný seznam
zviratka = ['opicka', 'orangutan', 'papousek', 'clovek']

Pokud chceš přidat něco nakonec seznamu, použij metodu .append().

zviratka.append('oslik')

Seznam může obsahovat jakýkoliv datový typ, případně i další seznam.
Pokud seznam jednou definuješ, vložíš do něj nějakou proměnnou a poté proměnnou změníš, v seznamu zůstane její starý obsah.

17. For cyklus

For je také cyklus, podobně jako while, ale má omezený počet opakování.
např.

seznamek = []

for i in range(20):
# i je proměnná, která se při každém opakování cyklu mění na čísla od 0 do 19
    seznamek.append(i)
    print(*seznamek)
# * nemusí být použitá, jen je poté výsledek čitelnější

Stejný seznam můžeš vytvořit takhle:

hokus_pokus = [i for i in range(20)]

18. Lokální proměnné

Lokální proměnná je taková proměnná, která definovaná na omezenou dobu. Když splní svůj účel, zmizí.
Takové proměnné najdeme třeba v cyklech a to konkrétně i ve for cyklu. Jsou definované pro každé jedno zopakování cyklu, ale když cyklus skončí, tak jsou znovu nedefinované.
To ale neplatí, pokud proměnnou definujeme už před začátkem cyklu:

for i in range(5):
    print(i)

print(i)

# vyskočí error, i is not defined
# pokud ale proměnnou i definujeme před for cyklem (např. i = None - None znamená, že proměnná bude existovat, ale bude prázdná), už z něj nebude lokální proměnná, ale globální proměnná a error nám nevyskočí.

19. For cyklus pro seznamy

For cyklus nemusíme použít jen na procházení po sobě jdoucích čísel, ale třeba i na procházení jednotlivých položek v seznamu.

to_do = ['ukol z matiky', 'koupit listek na ples', 'napsat si omluvenku', 'koupit darek fantovi']

for i in to_do:
    print(i)     if input('pokud to mas hotove napis ano, jinak napis ne') == 'ano':
        to_do.remove(i)

print('Tohle všechno ti zbývá dodělat: ', *to_do)

20. Index

Věci v seznamech jsou očíslované od 0. Takovému číslu se říká index.
Pokud chceme zavolat položku ze seznamu, a víme, na jaké pozici se nachází (známe její index), stačí napsat:

nazev_seznamu[index]

a vrátí se nám položka s tímto indexem.

obj_sou = ['b', 'f', 'l', 'm', 'p', 's', 'v', 'z']
obj_sou[5]    # vrátí 's'

22. Argument

Argument je parametr, který dáváš funkci, aby s tím mohla pracovat.

Příklady:

input() může vzít jeden argument (string), na který se zeptá dřív, než bude požadovat odpověď.
randint() vezme dva argumenty (itegery = celá čísla), aby věděl v jakém rozmezí má vrátit náhodné číslo.

Funkci s argumentem si také můžeš vztvořit sám:

def scitani(a, b):
    return a + b

scitani(input(), input())
#tahle funkce už existuje a to ve stylovějším provedení: sum()

23. argv

Pokud by si chtěl/a pythonem zpracovávat nějaký, třeba textový, soubor, lze to udělat několika způsoby.
Tady je jeden z nich.

Aby fungoval, musíš mít soubor, který chceš zpracovat, ve stejné složce jako svůj .py soubor.
Naše zaklínadlo bude vypadat nějak takhle:

from sys import argv
program_name, file_name = argv

file = open(file_name, 'r')

for i in file:
    print('délka řádku:', len(i))
#len() vrátí číslo - délku stringu nebo seznamu
